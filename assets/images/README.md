# UX Portfolio Website - Asset Guidelines

## Image Requirements

### Profile Image
- **Filename**: `andrew-profile.jpg`
- **Dimensions**: 400x400px (square aspect ratio)
- **Format**: JPG or PNG
- **Quality**: High resolution, professional headshot

### Project Images
All project images should be:
- **Dimensions**: 800x600px (4:3 aspect ratio)
- **Format**: JPG or PNG
- **Quality**: High resolution, showcasing the design work
- **Naming Convention**: `project-name.jpg`

#### Required Project Images:
1. `ecommerce-redesign.jpg` - E-commerce platform redesign
2. `mobile-banking.jpg` - Mobile banking app
3. `healthcare-dashboard.jpg` - Healthcare provider dashboard
4. `learning-platform.jpg` - Online learning platform
5. `smart-home.jpg` - Smart home control app
6. `food-delivery.jpg` - Food delivery app redesign

### Placeholder Image
- **Filename**: `placeholder-project.jpg`
- **Purpose**: Fallback image for projects without specific images
- **Dimensions**: 800x600px
- **Content**: Generic design-related placeholder

### Favicon
- **Filename**: `favicon.ico`
- **Dimensions**: 32x32px
- **Format**: ICO
- **Content**: Personal logo or initials "AL"

## Image Optimization Tips

1. **Compression**: Use tools like TinyPNG or ImageOptim to reduce file sizes
2. **WebP Format**: Consider using WebP for better compression (with JPG fallback)
3. **Responsive Images**: Provide multiple sizes for different screen densities
4. **Alt Text**: All images should have descriptive alt text for accessibility

## Adding New Project Images

When adding new projects:

1. Create a high-quality mockup or screenshot of your work
2. Resize to 800x600px
3. Optimize for web (aim for under 200KB)
4. Save to `/assets/images/projects/`
5. Update the project data in `js/projects.js`
